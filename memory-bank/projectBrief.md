# Project Brief: Amrit's Blog

## 🎯 High-Level Goals
- Build and maintain a personal blog site using Nikola.
- Host blog posts on various technical and lifestyle topics.
- Provide a clean, fast, and accessible reading experience.
- Enable easy deployment to GitHub Pages.

## 🛠️ Overview
Amrit's Blog is a static site generated from Markdown files using Nikola. It uses the `bootstrap4` theme and is configured for multilingual support (though currently primarily English).

## 🚀 Key Features
- Blog archive by month/year.
- Tags and categories for easy navigation.
- Responsive design via Bootstrap 4.
- Integration with external social platforms (YouTube, LinkedIn, GitHub).
- Search functionality via DuckDuckGo.
