# Product Context: Amrit's Blog

## 🌟 Why This Exists
This blog serves as a personal portfolio and knowledge-sharing platform for Amrit. It's a place to document learnings, share technical tips, and discuss various interests.

## 👤 Target Audience
- Fellow software developers.
- Readers interested in tech, productivity, and life.
- Potential employers and collaborators.

## 💡 How It's Used
- Writing: Posts are written as Markdown files in the `posts/` folder.
- Generation: Nikola compiles these into static HTML files.
- Hosting: The `output/` folder is typically pushed to the `master` branch of the `amritpandey23.github.io` repository for GitHub Pages hosting.
